# Flutter Polls

Customizable Polls for Flutter.
Simple, easy to use and highly customizable.

## Getting Started

In the `pubspec.yaml` of your flutter project, add the following dependency:

```yaml
dependencies:
  ...
  flutter_polls: ^0.0.1
```

Import it:

```dart
import 'package:flutter_polls/flutter_polls.dart';
```
